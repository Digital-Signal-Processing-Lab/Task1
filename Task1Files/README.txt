This is a template. You are welcome to edit it as per your wish.To make changes, go to the folder 'edit'.

1. Edit the file 'details.tex' to add your Name and Roll number

2. Make changes in the file 'content.tex' to add details of the experiments and figures

3. Place all the figures in the edit folder

4. All figures should be in .png format

5. Figure names better not have spaces in between (some times it creates problem)
